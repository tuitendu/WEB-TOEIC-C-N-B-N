# TOEIC Vocabulary Learning Website

## Overview

TOEIC Vocabulary Learning Website is a full-stack web application designed to support English learners in improving their TOEIC vocabulary efficiently. The platform provides an interactive learning environment where users can study vocabulary by topic, take quizzes, review incorrect answers, monitor learning progress, and manage their personal accounts.

The project is developed using a modern technology stack consisting of React for the frontend, FastAPI for the backend, and SQLite for data storage. Vocabulary data is organized in JSON format, making it easy to maintain and expand.

This project was created for educational purposes and software engineering practice while demonstrating the implementation of a complete web application following a client-server architecture.

# Features

## User Authentication

The application provides a complete authentication system.

Features include:

- User Registration
- User Login
- Session Management
- User Information Storage

Each user has an independent learning history and progress.

---

## Vocabulary Learning

The learning module allows users to study vocabulary based on TOEIC topics.

Each vocabulary item contains:

- English Word
- Vietnamese Meaning
- Phonetic Pronunciation
- Example Sentence
- Pronunciation Button
- Learning Status

Users can navigate through vocabulary easily while listening to native-like pronunciation generated by the browser.

---

## Topic-Based Learning

Vocabulary is categorized into multiple TOEIC topics to improve learning efficiency.

Examples include:

- Office
- Marketing
- Finance
- Travel
- Transportation
- Shopping
- Business
- Restaurants
- Meetings
- Housing
- Entertainment

Each topic contains a collection of related vocabulary words.

---

## Quiz Module

The quiz system helps learners evaluate their understanding through multiple-choice questions.

Main functions:

- Random question generation
- Random answer order
- Topic selection
- Immediate answer validation
- Automatic score calculation
- Quiz result storage
- Learning statistics

The quiz system encourages active recall, which improves vocabulary retention.

---

## Review Module

The review module helps users reinforce previously learned vocabulary.

Functions include:

- Review learned words
- Review incorrect answers
- Practice difficult vocabulary
- Repeat learning sessions

This module supports long-term memorization through repeated exposure.

---

## Progress Tracking

The system records user learning progress.

Stored information includes:

- Number of completed topics
- Number of learned vocabulary words
- Quiz scores
- Review history
- Learning statistics

Users can easily monitor their improvement over time.

---

## Pronunciation Support

Vocabulary pronunciation is generated using the browser Speech Synthesis API.

Advantages include:

- Native browser support
- No external API required
- Fast response
- High compatibility
- Better pronunciation practice

# System Architecture

The project follows a Client-Server architecture.

```
                +----------------------+
                |      React Client    |
                |  (Frontend - Vite)   |
                +----------+-----------+
                           |
                    REST API Request
                           |
                           v
                +----------------------+
                |      FastAPI         |
                |      Backend         |
                +----------+-----------+
                           |
             +-------------+--------------+
             |                            |
             v                            v
      TOEIC Vocabulary JSON          SQLite Database
```

# Installation

## Clone Repository

```bash
git clone https://github.com/tuitendu/WEB-TOEIC-C-N-B-N.git
```

Move to project directory.

```bash
cd WEB-TOEIC-C-N-B-N
```

---

# Backend Installation

Navigate to backend.

```bash
cd backend
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the server.

```bash
uvicorn main:app --reload
```

Backend Address

```
http://127.0.0.1:8000
```

Swagger Documentation

```
http://127.0.0.1:8000/docs
```

---

# Frontend Installation

Navigate to frontend.

```bash
cd frontend
```

Install packages.

```bash
npm install
```

Run development server.

```bash
npm run dev
```

Frontend Address

```
http://localhost:5173
```

# Vocabulary Dataset

The application uses a TOEIC vocabulary dataset containing approximately 600 commonly used TOEIC words.

Each vocabulary entry contains:

```json
{
    "word": "apply",
    "meaning": "nộp đơn",
    "phonetic": "/əˈplaɪ/",
    "example": "She applied for a new position."
}
```

Dataset location:

```
data/toeic600.json
```
